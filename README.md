# Amazon Webpage Clone

This project is a front-end **Amazon homepage clone** created using **HTML and CSS**.  
The purpose of this project is to practice **layout design, flexbox, grid, and responsive UI structure**.

## 🚀 Features
- Amazon-style homepage layout
- Header, navbar, banner, and product sections
- Clean and structured UI
- Fully responsive design (basic)
